# Ejecutra en desarrollo

1. Clonar repositorio
2. Ejecutar

   `pnpm install`

3. Tener nest CLI instalado
   `pnpm i -g @nestjs/cli`
4. Levantar BBDD
   `docker-compose up -d`

## Stack

- MongoDB
- NestJS
